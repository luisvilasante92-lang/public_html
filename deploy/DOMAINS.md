# Настройка доменов: luisv-records.ru

Вход в личный кабинет: **https://auth.luisv-records.ru** (страница логина).  
После входа пользователь попадает на **https://lk.luisv-records.ru** (приложение ЛК).

## Что сделано в коде

- Все URL в конфигурации переведены на домен **luisv-records.ru** и его поддомены:
  - `auth.luisv-records.ru` — вход в личный кабинет (страница логина)
  - `lk.luisv-records.ru` — личный кабинет (приложение после входа)
  - `lnk.luisv-records.ru` — смарт-ссылки
- В конфиг добавлена переменная **APPLICATION_ROOT_URL** (по умолчанию `https://luisv-records.ru`) для единого домена в ссылках.
- В production включён **ProxyFix**: приложение доверяет заголовкам `X-Forwarded-Host`, `X-Forwarded-Proto` от обратного прокси, чтобы формировать правильные абсолютные ссылки (в письмах, редиректах).

## Что нужно сделать на хостинге

### 1. Кнопка «Войти» на лендинге

На основном сайте **luisv-records.ru** кнопка «Войти» должна вести на:

- **https://auth.luisv-records.ru** (или **https://auth.luisv-records.ru/login**)

Измените ссылку в шаблоне/редакторе лендинга.

### 2. Настройка поддоменов на хостинге

- **auth.luisv-records.ru** — укажите корневую папку приложения (System_LK1). Здесь открывается страница входа.
- **lk.luisv-records.ru** — та же корневая папка приложения. После входа пользователь попадает сюда. Смарт-ссылки тоже работают на этом домене: `https://lk.luisv-records.ru/link/КОД`.

Оба поддомена могут указывать на один и тот же каталог с приложением; различие только в том, с какого URL пользователь заходит (auth — для входа, lk — для работы в ЛК).

**Опционально**: поддомен **lnk.luisv-records.ru** — если хотите короткие ссылки вида `lnk.luisv-records.ru/link/КОД`, создайте DNS A‑запись для lnk и настройте виртуальный хост, указывающий на тот же каталог приложения. Затем в `.env` задайте `SMART_LINK_BASE_URL=https://lnk.luisv-records.ru/link`.

### 3. Переменные окружения (.env)

На сервере приложения в `.env` заданы:

- `AUTH_SERVICE_URL=https://auth.luisv-records.ru`
- `AUTH_CALLBACK_URL=https://lk.luisv-records.ru/auth/callback`
- `APPLICATION_ROOT_URL=https://luisv-records.ru` — основной домен в ссылках (по желанию).

После изменений перезапустите приложение.
